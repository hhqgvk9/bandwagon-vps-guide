## 搬瓦工 VPS 是什么，适合哪些人

VPS 就是虚拟私有服务器（Virtual Private Server）的缩写——你可以把它理解成放在云端的一台 Linux 小机器，完整的系统，你有 root 权限，想装什么装什么，和本地操作没啥大区别。

搬瓦工走的是自托管路线，不帮你管服务器，你得自己搞定一切。这一点让价格压下来了，也让它更适合有点动手能力的人。

适合买瓦工 VPS 的场景：

- 个人建站、博客、小工具，预算有限又想要稳定线路
- 学 Linux、搭开发环境、跑测试服务
- 需要稳定的境外节点，对延迟有一定要求
- 部署 AI 应用（n8n、Dify 等），现在用搬瓦工跑这些的人不少
- 外贸业务，需要一台国外机器做跳板或加速

不太适合的：

- 需要人帮你装软件、处理问题——客服是英文工单，响应不慢，但不会手把手帮你操作
- 对 CPU 密集型任务有高要求——它是共享宿主机的，CPU 性能不是重点
- 需要大量带宽做 CDN 或高防——CN2 GIA 遇到 DDoS 会临时封 IP，不适合

## 搬瓦工线路科普：CN2 GT、CN2 GIA、GIA-E 有什么不同

这块不搞清楚，选套餐就容易踩坑。

**CN2 GT（Global Transit）**：电信二级线路，早年还行，2019 年以后拥堵越来越严重，现在高峰期体验和普通线路差不多。搬瓦工官方已经停售这条线路的独立套餐了。

**CN2 GIA（Global Internet Access）**：电信最顶级的 IP 中转线路，带宽单独保障，高峰期几乎不丢包。好用在哪？简单说就是别人家晚上网速掉一半，GIA 照常跑。代价是贵——市场批发价每 Mbps 能到 120 美元，一般服务商根本拿不到这条线路。

**CN2 GIA-E（ECOMMERCE）**：这是搬瓦工对 CN2 GIA 的封装套餐名，也是目前性价比最高的选择。买一个套餐，开通后可以在 12-13 个机房之间自由迁移，包括洛杉矶 DC6、DC9、日本大阪软银、荷兰联通 9929 等，哪个机房速度好就切哪个。

三网线路说明（以 DC9 USCA_9 为例）：

- 电信：去程+回程全程 CN2 GIA
- 联通：联通 Premium（AS10099）
- 移动：CMIN2（中国移动国际精品网络）

讲真，这三条线路同时跑起来，绝大多数国内用户的体验都不会差。

## 几个主要机房，延迟和线路说清楚

搬瓦工的卖点之一就是机房多，CN2 GIA-E 套餐开通后可以在控制面板里一键迁移，不用重新买。以下是国内用户比较关心的几个机房：

**洛杉矶 DC6 CN2 GIA-E（USCA_6）**：搬瓦工最经典的机房，三网优化，延迟在 150-180ms 左右。电信联通走 CN2 GIA，移动走 CMIN2，高峰期表现稳定，是 CN2 GIA-E 套餐默认机房之一。

**洛杉矶 DC9（USCA_9）**：搬瓦工自己说这个机房网络容量和稳定性最强，三网都走顶级线路，还已经完成 AMD EPYC + NVMe 升级。用下来这个机房比 DC6 还要稳一些，延迟差不多，主要是带宽容量更大。

**日本大阪软银（JPOS_1）**：延迟在 40-80ms，是 CN2 GIA-E 套餐里速度最快的机房之一。软银线路对移动用户特别友好，三网表现都不错，是目前性价比很高的一个选择。

**香港 MEGA2（HKHK_8）**：物理距离最近，广东方向延迟能低到 10ms 以内，全程 CN2 GIA。就是贵，月付 $89.99 起，适合对延迟极度敏感的场景（实时通讯、在线游戏加速等）。

**荷兰阿姆斯特丹（EUNL_9）**：联通 9929 线路，欧洲访问速度好，国内联通用户体验也不差。有海外用户或欧洲业务需求的可以考虑迁过去试试。
